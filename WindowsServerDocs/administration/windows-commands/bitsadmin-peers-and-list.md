# <a name="bitsadmin-peers-and-list"></a>bitsadmin peers e list

Elenca tutti i peer.

## <a name="syntax"></a>Sintassi

```
bitsadmin /peers /list
```

## <a name="examples"></a>Esempi

Per elencare tutti i peer:

```
bitsadmin /peers /list
```
