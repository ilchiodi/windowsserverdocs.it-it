# <a name="bitsadmin-peers-and-clear"></a>bitsadmin peers e clear

Cancella l'elenco di peer.

## <a name="syntax"></a>Sintassi

```
bitsadmin /peers /clear
```

## <a name="examples"></a>Esempi

Per cancellare l'elenco di peer.

```
bitsadmin /peers /clear
```
