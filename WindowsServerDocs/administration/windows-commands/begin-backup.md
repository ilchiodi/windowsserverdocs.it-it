# <a name="begin-backup"></a>Avvia backup

avvia una sessione di backup completo.

## <a name="syntax"></a>Sintassi

```
begin backup
```

## <a name="remarks"></a>Note

Questo comando sostituisce l'impostazione predefinita di backup di copia.
