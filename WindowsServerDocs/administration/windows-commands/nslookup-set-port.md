# <a name="nslookup-set-port"></a>nslookup set port

Consente di modificare la porta predefinita del server dei nomi TCP/UDP Domain Name System (DNS) al valore specificato.

## <a name="syntax"></a>Sintassi

```
set port=<port>
```

### <a name="parameters"></a>Parametri

| Parametro | Descrizione |
| ---------- | ---------- |
| `<port>` | Specifica il nuovo valore per la porta predefinita del server dei nomi DNS TCP/UDP. La porta predefinita Ã¨ **53**. |
| /? | Visualizza la guida al prompt dei comandi. |
| /help | Visualizza la guida al prompt dei comandi. |
