# <a name="bitsadmin-cache-and-clear"></a>bitsadmin cache e clear

Elimina la cache locale.

## <a name="syntax"></a>Sintassi

```
bitsadmin /cache /clear
```

## <a name="examples"></a>Esempi

Per eliminare la cache locale:

```
bitsadmin /cache /clear
```
