# <a name="checklist-setting-up-a-federation-server"></a>Elenco di controllo: configurazione di un server federativo

Questo elenco di controllo include le attività di distribuzione necessarie per preparare un server che esegue Windows Server® 2012 per il ruolo server federativo in Active Directory Federation Services \(AD FS\).  
  
> [!NOTE]  
> Completare le attività dell'elenco di controllo nell'ordine indicato. Quando un collegamento a un riferimento porta a una procedura, tornare a questo argomento una volta completati i passaggi della procedura, in modo da poter procedere con le operazioni rimanenti nell'elenco di controllo.
